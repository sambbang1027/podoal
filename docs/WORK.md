## 진행 예정

### 다음 작업 — Zone 봇 타이밍 현실화

#### 문제
VIP가 약 2~3초 만에 매진되는데, 2층 봇 시작 딜레이는 4,000ms.
→ VIP 매진 시점에 2층은 한 자리도 빠지지 않은 상태 — 비현실적.

실제 티켓팅: 대부분이 VIP를 노리지만, 일부는 처음부터 2층/3층을 노린다.
즉, 2층/3층도 VIP 매진 전부터 어느 정도 소진되고 있어야 자연스럽다.

#### 해결 방향 (논의 필요)

**옵션 A — startDelay 단축**
- 2층: 4000ms → 1500ms, 3층: 8000ms → 3000ms
- VIP 매진 전에 이미 2층/3층도 줄기 시작하는 느낌

**옵션 B — VIP 매진 시 2층 즉시 오픈**
- VIP 좌석 0 감지 → `openedZones.floor2 = true` + 봇 즉시 시작
- 2층이 VIP 빠진 사람들을 흡수하는 느낌 (현실 고증에 가까움)

**옵션 C — 혼합**
- 2층은 2초 후 자동 시작 (소수 초반부터 노림)
- VIP 매진 시 2층 봇 추가 투입 (러시 시뮬레이션)
